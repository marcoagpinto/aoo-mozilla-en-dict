Legacy Version (for older applications)
---------------------------------------
For Mozilla Firefox or Mozilla Thunderbird version 59 and
below, download the legacy .xpi by clicking the “Download”
button. ⚠️

After installing, disable automatic updates to prevent the
dictionary from being upgraded to the WebExtension version.

You will need to update it manually in the future.


Supported applications
----------------------
The legacy add-on supports at least the following:
 - Mozilla Firefox
 - Firefox for Android
 - Mozilla Thunderbird
 - SeaMonkey
 - Epyrus
 - Pale Moon
 - Waterfox
